# EmmyvibezLogs Marketplace

**Description:**  
EmmyvibezLogs is a marketplace platform for buying and selling social media accounts and services, focused on creators, businesses, and tech startups in Nigeria. Built with React, Node.js, Express, MongoDB, and Paystack for payments.

---

## Features
- Buyer and Seller account registration
- Business-name based profiles
- Fixed and package services
- Orders with escrow system
- Automatic payouts with 15% platform commission
- Admin dashboard to approve services and resolve disputes
- Fully responsive frontend (React + Tailwind)

---

## Technologies
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **Payments:** Paystack (Nigerian users)
- **Deployment:** Render (backend), Vercel (frontend)
